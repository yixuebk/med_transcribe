# Test Results
Test results for the files included in the repo. Test audio is generated using ElevenLabs, using example transcripts generated by ChatGPT.

## Processing Times
- OpenAI models test via OpenAI API.
- Local model transcription uses the Python ```faster-whisper``` library, and using the faster-whisper-large-v3-turbo model automatically downloaded from HuggingFace.
- Local models for summarization are hosted using LM Studio local server API, with GPU options turned off for CPU-only testing.
